# SOC: Containment

SOC: Containment é um jogo de estratégia, investigação e gerenciamento de incidentes de segurança cibernética. A rede corporativa é o campo principal: o jogador observa sua estrutura, entende a função de cada ativo e toma decisões para preservar a operação.

## Como funciona

O mapa apresenta dispositivos e conexões de uma pequena rede corporativa. Cada ativo pode ser selecionado para exibir seu contexto operacional, como categoria, função, zona de rede, endereço, descrição e importância.

## Funcionalidades principais

- Mapa de rede navegável com zoom e pan.
- Dispositivos com representações visuais distintas e feedback de hover e seleção.
- Conexões entre ativos e tráfego corporativo visual em tempo real.
- Painel de detalhes operacionais do dispositivo selecionado.
- Controle para restaurar a visualização inicial do mapa.
- Relógio de simulação com pausa e velocidade 2×.
- Primeiro alerta roteirizado, com observações recentes e inspeção de processos.

## Requisitos e execução

Requer Godot 4.7. Abra `project.godot` no editor e execute a cena principal.
