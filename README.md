# SOC: Containment

Uma fundação visual para um jogo de estratégia, investigação e gerenciamento de incidentes de segurança cibernética.

Nesta etapa, o projeto apresenta uma pequena rede corporativa interativa: dispositivos selecionáveis, conexões, tráfego visual legítimo, painel de detalhes e navegação por zoom e pan.

Requer Godot 4.7. Abra `project.godot` no editor e execute a cena principal.

O projeto está em desenvolvimento.
